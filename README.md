# Retroslate

Primary company website for Retroslate.

## Stack

- Next.js App Router
- TypeScript
- Vercel-ready deployment

## Local Development

```bash
npm install
npm run dev
```

The site defaults to the dark theme using the locked Retroslate palette:

- Void: `#23212c`
- Wine: `#7f011f`
- Milk: `#f8f5f2`
